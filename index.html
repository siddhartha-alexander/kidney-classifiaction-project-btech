Project Overview

Fast, consistent support for CT kidney diagnosis

Problem

Manual CT scan interpretation is time-consuming and can vary between observers. Earlier CAD systems often depend on handcrafted features and struggle with complex multi-class medical image patterns.

Proposed Solution

The system fine-tunes a pretrained ResNet18 convolutional neural network on the CT Kidney Dataset from Kaggle to classify images into four clinically relevant groups.

Impact

The classifier can help healthcare professionals screen kidney CT images more efficiently while reducing delay in early disease identification.

12,446 CT images
4 classes
80 / 20 train-validation split
ResNet18 transfer learning model

Methodology

From CT image to disease class

01

Dataset Collection

CT Kidney Dataset with Cyst, Normal, Stone, and Tumor image categories.

02

Preprocessing

Images are resized and normalized before being passed into the model.

03

Augmentation

Random horizontal flipping and rotation improve robustness and reduce overfitting.

04

Model Training

ResNet18 is fine-tuned using Adam optimizer and cross-entropy loss for five epochs.

05

Evaluation

Accuracy, confusion matrix, precision, recall, and F1-score are used for validation.
